# はじめに

**quantum-simpl** は、ハミルトニアンのシンボリック前処理ツールです。簡潔なモデル定義から [mVMC](https://github.com/issp-center-dev/mVMC) の入力ファイルを自動生成します。

## 背景

[mVMC](https://github.com/issp-center-dev/mVMC) は量子格子模型の変分モンテカルロソルバーです。ハミルトニアンを定義するために `Trans.def`、`InterAll.def` などの構造化された入力ファイルが必要ですが、大規模なモデルではこれらを手作業で書くのは煩雑でミスが起きやすい作業です。

## quantum-simpl の役割

quantum-simpl はこの問題を解決します。独自のDSL（ドメイン固有言語）で短いモデル定義を書くだけで、以下の5段階のパイプラインを通して全ての mVMC 入力ファイルを生成します：

1. **展開** -- sumループの展開、エルミート共役の展開、`n(i,s)` を `c†(i,s) c(i,s)` に脱糖
2. **正規順序化** -- フェルミオンの反交換関係とスピンの交換関係を適用し、生成演算子を消滅演算子の左に配置
3. **結合** -- 同一演算子列の重複除去と係数の和
4. **Szフィルタ** -- Sz保存を破る項の除去
5. **出力** -- mVMC入力ファイルの書き出し（`namelist.def`、`trans.def`、`interall.def` など）

**Hubbard模型**と**Heisenberg模型**の両方に対応しているほか、スピン相関関数や密度相関関数の測定用入力ファイルを生成する**相関関数パイプライン**も備えています。
