# DSL構文

quantum-simpl のDSL（ドメイン固有言語）は、量子格子模型のハミルトニアンを簡潔に記述するために設計されています。

## 全体構造

入力ファイルは3つのセクションで構成されます：

```
lattice 1d sites=N pbc=true|false

sum var=start..end:
  項の定義...

params:
  パラメータ = 値
```

## lattice 宣言

格子の次元、サイト数、境界条件を指定します。

```
lattice 1d sites=10 pbc=true
```

- `1d` -- 1次元格子
- `sites=N` -- サイト数
- `pbc=true|false` -- 周期的境界条件の有無

## sum ブロック

sumブロックはループ変数と範囲を指定し、ハミルトニアンの項を定義します。範囲 `start..end` は**両端を含みます**。

```
sum i=0..9:
  -t * c†(i,up) c(i+1,up) + h.c.
  U * n(i,up) n(i,down)
```

### エルミート共役

`+ h.c.` を行末に付けると、その項のエルミート共役が自動的に追加されます。

### 複数のsumブロック

複数のsumブロックを定義できます：

```
sum i=0..9:
  -t * c†(i,up) c(i+1,up) + h.c.

sum i=0..9:
  U * n(i,up) n(i,down)
```

## 演算子

### フェルミオン演算子

| 演算子 | 構文 | 説明 |
|--------|------|------|
| 生成演算子 | `c†(i,spin)` | フェルミオン生成演算子 |
| 消滅演算子 | `c(i,spin)` | フェルミオン消滅演算子 |
| 数演算子 | `n(i,spin)` | `c†(i,s) c(i,s)` の糖衣構文 |

スピンの値: `up`、`down`

### スピン演算子

| 演算子 | 構文 | 説明 |
|--------|------|------|
| スピン上昇 | `Sp(i)` | `c†(i,up) c(i,down)` に変換 |
| スピン下降 | `Sm(i)` | `c†(i,down) c(i,up)` に変換 |
| スピンz | `Sz(i)` | `0.5*n(i,up) - 0.5*n(i,down)` に変換 |
| スピン内積 | `S(i) . S(j)` | `0.5*Sp(i)Sm(j) + 0.5*Sm(i)Sp(j) + Sz(i)Sz(j)` に展開 |

### インデックス式

インデックスには以下の形式が使えます：

- ループ変数: `i`
- オフセット付き: `i+1`、`i-1`
- リテラル整数: `0`、`5`

## params ブロック

ハミルトニアン中のパラメータに数値を代入します。

```
params:
  t = 1.0
  U = 4.0
```

## 開放境界条件に関する注意

`pbc=false` を使用する場合、格子範囲外のサイトを参照する項は警告付きで除去されます。sumの範囲が格子サイズと整合するよう注意してください。

例えば `sites=10 pbc=false` で最近接ホッピングを定義するには、`sum i=0..8` を使用してください。`i=9` では `c†(9,s) c(10,s)` が生成され、サイト10は範囲外となります。
